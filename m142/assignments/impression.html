Impressionism

In this assignment, you will generate drawings using randomly placed smudges whose colors are determined by an image of your choosing.

Turning it: To turn in this project, you should place the website in a folder called impression inside of your class website and link the site from your main class homepage.

Requirements:

The site should contain at least five images, one drawn with each of these types of marks: random line segments, random line segments following a noise pattern, Brownian motion, projectile motion, and noise hair. If you like you can place all of these images on one page with some text describing each one. Alternatively, you can create a photo gallery of the images.

Your five (or more) images do not all have to be based on the same image.

If you would like to include a link to a spork page which actually does the drawing (which you must have constructed to get your drawings) you may.
