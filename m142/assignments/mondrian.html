Mondrian Grid

Create a website that illustrates the use of the noise function to draw shapes and patterns.

Turning it: To turn in this project, you should place the website in a folder called mondrian inside of your class website and link the site from your main class homepage.

Requirements: The site should contain at least these pages:
  • (1 page) A home page with a title and links to the pages listed below. Each page below should include a Spork canvas on which your image is drawn.
  • (1 page) Draw a Mondrian grid in which every rectangle is filled with a random color.
  • (1 page) Draw a Mondrian grid in which every rectangle is filled with a color determined by an image as you did in the Impressionism assignment.
  • (1 page) Draw a Mondrian grid; however, instead of drawing rectangles draw some other geometric shape. Use your imagination.
