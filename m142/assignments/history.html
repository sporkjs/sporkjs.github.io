Math History Assignment

Write a historical report about a mathematician in the form of a website.

Turning it: To turn in this project, you should place the website in a folder called history inside of your class website and link the site from your main class homepage.

Requirements:
  • The site should contain at least these pages:
    • An index page with an overview and biographical data.
    • At least two pages devoted to mathematical, scientific, or philosophical contributions.
    • A reference page.
  • Each page except possibly the reference page should include an image.
  • The entire site should be styled uniformly using a single external css file.
  • Each page should include: a wrapper div, a title div, a main navigation div, a content div, and a secondary navigation div (the standard containers).
