Implicit Functions

Create a website that illustrates the use of implicit lines and circles to draw an image.

Turning it: To turn in this project, you should place the website in a folder called implicit inside of your class website and link the site from your main class homepage.

Requirements:
  • Each of the main pages below can be based on the same underling image. Your base image should use at least three implicit curves. This should include at least one line and at least one circle.
  • The site should contain at least these pages:
    • (1 page) A home page with a title and links to the pages listed below. Each page below should include a Spork canvas on which your image is drawn.
    • (1 page) A page with your image drawn with regular geometric figures placed on a grid.
    • (1 page) A page with your image drawn with regular geometric figures placed randomly.
    • (1 page) A page with your image drawn with blotches placed on a grid. For blotches, you can use blotches like we did in class, or you can use your Brownian motion or mark functions from the motion assignment.
  • For the randomly placed marks, you can use a for loop with a set number of shapes, or you can use the animation loop. If you use the animation loop, be sure to include a buttons to Stop, Go, and Start Over.
