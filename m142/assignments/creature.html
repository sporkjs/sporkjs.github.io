Creature Assignment

Use the Spork drawing commands to draw a creature. This creature will serve to identify you on course web pages, so take some time to personalize it. Save the code from this assignment because you will need it later.

Turning it: To turn in this project, email me your Spork code.

Requirements:
  • Use the standard 400x400 Spork canvas.
  • Your first command MUST BE translate(200,200);. This is very important for how we will use this code later.
  • Try to fill the entire canvas (with x and y values close to 0 and close to 400).
  • Do not color the background.
  • Use at least three drawing shapes.
  • Use at least three colors.
