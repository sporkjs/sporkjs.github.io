Noise Gallery

Create a website that illustrates the use of the noise function to draw shapes and patterns.

Turning it: To turn in this project, you should place the website in a folder called noise inside of your class website and link the site from your main class homepage.

Requirements: The site should contain at least these pages:
  • (1 page) A home page with a title and links to the pages listed below. Each page below should include a Spork canvas on which your image is drawn.
  • (1 page) Hair On this page, you will modify the Brownian motion page from your Motion assignment by following these steps:
    • Add a setupNoise(5); command to your setup() function. You can experiment with numbers other than 5 later.
    • Your brownian(x,y) function should have a line of the form angle=random()*360;. Change this to angle=noise(x/width,y/height)*360;
    • As it is, this page will occasionally give an error. The reason for the error is that the noise() function wants arguments between 0 and 1. The values of x and y may be too big or small for this, so we need to make the points wrap around to stay on the canvas. You can do this like we did in the last two lines of the Wrap Around example.
  • (1 page) Circle Grid On this page, we will modify the Circle Grid Example by following these steps to make the radii of the circles determined by the noise function.
    • Add a setupNoise(5); command to your setup() function. You can experiment with numbers other than 5 later.
    • Inside the draw() function, add a declarion for a variable c which will (almost) be a noise value.
    • Before the command circle(x,y,20);, let c be a noise value by calling the command c=noise(x/width, y/height);
    • The variable c now has a value between -1 and 1. We want c to be between 0 and 1 so add this line: c=(c+1)/2;.
    • Now change circle(x,y,20); to circle(x,y,c*20);
    • Experiment with values other than 20.
    • Make a page that shows off the drawings that result.
  • (1 page) Funky Stripes On this page, we will modify the Noise Static Example by following these steps to change how we select the color:
    • For this example, we just want to round the gray level c to the nearest multiple of 20. This can be accomplished by adding the line c=20*floor(c/20); after the line c=floor(128*(noise(x/width, y/height)+1));
    • After you have a drawing working, experiment with values other than 20.
    • Make a page that shows off the drawings that result.
  • (1 page) Cow Spots On this page, we will modify the Noise Static Example by following these steps to change how we select the color:
    • Replace the line c=floor(128*(noise(x/width, y/height)+1)); with simply c=noise(x/width, y/height);. This just sets c to be a noise value.
    • Delete the line fillcolor(c,c,c);
    • In place of the line that you just deleted, place an if statement. If c>0, set the fillcolor to black; otherwise, set the fillcolor to white.
    • Experiment with comparing c to numbers other than 0.
    • Make a page that shows off the drawings that result.
