Noise Gallery

Create a website that illustrates the use of the noise function to draw shapes and patterns.

Turning it: To turn in this project, you should place the website in a folder called noise inside of your class website and link the site from your main class homepage.

Requirements: The site should contain at least these pages:
  • (1 page) A home page with a title and links to the pages listed below. Each page below should include a Spork canvas on which your image is drawn.
  • (7 pages) Include a page for each of these exercises from the in-class activity with noise: random static, smooth static, funky lines, cow spots, hair, wavy lines, circle grid.
