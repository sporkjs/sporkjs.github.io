Photo Gallery

ASSETS FOR THE CLASSROOM EXAMPLE

Construct a photo gallery similar to the one we did in class.

Requirements:
  • The site should include at least eight images.
  • Each image should be accompanied by a caption.
  • The images should be of artwork which has a mathematical influence.
  • Buttons for the images should use thumbnail images no larger than 200x200.
Turning it: To turn in this project, you should place the website in a folder called gallery inside of your class website and link the site from your main class homepage.
