Symmetric Noise Gallery

Create a website that illustrates the use of the noise function to draw shapes and patterns.

Turning it: To turn in this project, you should place the website in a folder called symmetry inside of your class website and link the site from your main class homepage.

Requirements: The site should contain at least these pages:
  • (1 page) A home page with a title and links to the pages listed below. Each page below should include a Spork canvas on which your image is drawn.
  • (1 page) Modify the Funky Stripes component of the Noise assignment in the following way to introduce reflection symmetry.
    • In this example, we will draw Funky Stripes with a copy reflected in the line y=x. To do so, we need first draw only those points on one side of the line y=x. To do so, we modify the for loop. Change the line for (y=0;y<height;y=y+1) to for (y=x;y<height;y=y+1). This will only give points above the line y=x.
    • To impose symmetry, every point we draw we must draw twice - once above the line y=x and once below. After the line fillrect(x,y,1,1); add fillrect(y,x,1,1);
    • You should now have symmetric funky lines.
  • (1 page) Modify the Cow Spots component of the Noise assignment in the same way as above to introduce reflection symmetry.
  • (1 page) Modify the Funky Stripes component of the Noise assignment in the following way to introduce symmetries about the x and y axes.
    • In the setup() function, add translate(width/2, height/2); so that the origin is at the center of the canvas.
    • Replace every command of the form fillrect(x,y,1,1); with four commands like we did in the Snowflakes example. The four commands should be
      fillrect(x,y,1,1);
      fillrect(-x,y,1,1);
      fillrect(x,-y,1,1);
      fillrect(-x,-y,1,1);
  • (1 page) Modify the Cow Spots component of the Noise assignment in the same way as above to introduce symmetries about the x and y axes.
  • (1 page) Modify the Hair component of the Noise assignment in the same way as above to introduce symmetries about the x and y axes.
