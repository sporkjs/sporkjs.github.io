Motion

Create a website that illustrates the use of the animation loop to model motion.

Turning it: To turn in this project, you should place the website in a folder called motion inside of your class website and link the site from your main class homepage.

Requirements: The site should contain at least these pages:
  • (1 page) A home page with a title and links to the pages listed below. Each page below should include a Spork canvas on which your image is drawn.
  • (1 page) On this page, you will draw a fuzz ball by modifying the Random Acceleration example. To do so, you will need to:
    • Add a loop(); command to your setup() function.
    • In your draw() function, add a small rotation before you start drawing. Do not save or restore.
    • In your setup() function, you might want to set the color to have some transparency so that you can see more "fuzziness."
    • Experiment with rotation amounts, accelerations, transparencies, colors, and the number of steps in the path.
    • Place the resulting code in a web page that includes buttons to Stop, Go, and Start Over.
  • (1 page) Make a page that has your creature bouncing with gravity in the Spork canvas. Include buttons for Stop, Go, and Start Over.
  • (1 page) Modify the Spots example so that at each random point (x,y) you draw the largest circle centered at that point that lies entirely on the canvas. The radius of this circle should be min(x, y, width-x, height-y);. Include buttons for Stop, Go, and Start Over.
  • (1 page) Select at least one of these options:
    • Convert the Random Mark example to a function mark(x,y) which places a random mark at (x,y).
    • Convert the Brownian Motion example to a function mark(x,y) which places a Brownian motion path at (x,y).
    Once you have selected an option, create a page that repeatedly calls mark(x,y); with random x and y. If you like, you can include random colors. As usual, include buttons for Stop, Go, and Start Over.
