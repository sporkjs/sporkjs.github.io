Math 142, Fall 2013

Assignment: In this project, we were asked to write code that would take a text input from the user along with a color choice, using a radio input. When the button is clicked, the Spork div is cleared, the Spork color is set, and the text will appear in the Spork div and the color chosen by the user.


Tools: In order to complete this, we used the Spork div, text input, radio input, color names, hidden input, and button functions from JavaScript programming and the Spork editor to make it come to life.
