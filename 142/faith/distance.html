Distance Function

Faith Hoppen

Math 142, Fall 2013

Assignment: In this project, we were asked to write code that will prompt the user to input the x and y values of two coordinate points. Then our program would calculate the distance between the two points with the formula (x1-y1)^2+(x2-y2)^2. Lastly, the user would be alerted with the the distance.


Tools: In order to complete this, we used basic JavaScript programming functions and the variable, prompt, alert, and eval functions from the Spork library.


Code:

//Faith Hoppen
//Math 142, Fall 2013
//Distance

function distance(x1,x2,y1,y2){
return(sqrt(((x1-y1)*(x1-y1))+((x2-y2)*(x2-y2))));
}
