Complementary Color Display

Faith Hoppen

Math 142, Fall 2013

Assignment: For this project, we were asked to create prompts that ask the user for red, blue, and green RGB values. Then using that RGB value we colored the background. The circle in the middle is the complement value of our original RGB. To find that value we subtracted the values given by the user from 255.


Tools: In order to complete this, we used basic JavaScript programming functions, prompt, alert, eval, and subtraction functions from the Spork library.


Code:

//Faith Hoppen
//Math 142, Fall 2013
//Complements

r=eval(prompt("RED: Enter a number from 0-255 for the amount of red.",''));

g=eval(prompt("GREEN: Enter a number from 0-255 for the amount of green",''));

b=eval(prompt("BLUE: Enter a number from 0-255 for the amount of blue.",''));

background(r,g,b);

red=255-r;
green=255-g;
blue=255-b;

circle(200,200,175);
fillcolor(red, green, blue);
fill();
