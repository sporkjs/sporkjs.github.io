Triangle Translation

Faith Hoppen

Math 142, Fall 2013

Assignment: In this project, we were asked to draw a right triangle similar to a picture with the labels lying on each side.


Tools: In order to complete this, we used basic JavaScript programming functions and the Spork library.

Picture:

Code:

//Faith Hoppen
//Math 142
//Triangle Transformation

background(204,153,255);

triangle(125,300,275,300,275,100);     //Triangle
fillcolor(153,153,255);
fill();

font("20px arial narrow")                  //Words
fillcolor(204,0,204);

save();
translate(145,222);
rotate(308);
filltext('hypotenuse',0,20);
restore();

save();
filltext('adjacent',172,319);
restore();

save();
translate(380,80);
rotate(90);
filltext('opposite',100,100);
restore();
