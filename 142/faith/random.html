Random Shapes

Faith Hoppen

Math 142, Fall 2013

Assignment: In this project, we were asked to write a function that would draw random circles(of radius 10), squares(side lengths 10), or triangles of random colors when the button was clicked.


Tools: In order to complete this, we used the variable, function, button, and random from JavaScript programming and the Spork editor to make it come to life.
