Flashing Shapes

Faith Hoppen

Math 142, Fall 2013

Assignment: In this project, we were asked to write a function, similar to examples shown, that would draw a series of the same objects that would randomly change.


Tools: In order to complete this, we used the variable, function, button, random, rotate, and loop commands from JavaScript programming and the Spork editor to make it come to life.
