Color

Faith Hoppen

Math 142, Fall 2013

Assignment: For this project we made a geometric picture based on ideas from three pictures from famous artists. In doing this, we incorporated some transparency and our own color themes.


Tools: In order to complete this, we used basic JavaScript programming functions and the Spork library.
