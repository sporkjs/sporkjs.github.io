Quilt

Faith Hoppen

Math 142, Fall 2013

Assignment: For this project we chose one of four traditional quilt block patterns, then we tanslated the origin of the canvas to the center and used the translation and rotation tools to complete the image.


Tools: In order to complete this, we used basic JavaScript programming functions and the translation, rotation, triangle, and color tools from the Spork library.
