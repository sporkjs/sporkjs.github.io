Blocky thing

Faith Hoppen

Math 142, Fall 2013

Assignment: In this project, we were given only the parameters of an olive background and a line thickness of 30px. Then we were asked to create a simple block-like shape.


Tools: In order to complete this, we used basic JavaScript programming functions and the Spork library.

Picture:

Code:

// Faith Hoppen
// Math 142
// Blocky thing 

background('olive');        //Olive background
linewidth(30);              //Line width 30 px
line(30,200,370,200);       //Horizontal line
line(200, 120, 200, 270);   //Vertical line
line(100,30,100,370);
line(300,30,300,370);
line(30,200,100,30);        //Slanted line
line(300,370,370, 200);
line(100, 30, 300, 30);
line(100,370,300,370)
