Draw on Command

Faith Hoppen

Math 142, Fall 2013

Assignment: In this project, we were asked to write a function that would draw a picture when the button was clicked.


Tools: In order to complete this, we used the variable, function, and button from JavaScript programming and the Spork editor to make it come to life.


Code:


//Faith Hoppen
//Math 142, Fall 2013
//Draw on command


function daisy(){
  size(400,400);
  translate(200,200);
  rotate(15);
  background(0,51,102);
  color('white');
  ellipse(0,-100,20,90);
  fillcolor('white');
  fill();
  ellipse(0,100,20,90);
  fillcolor('white');
  fill();
  ellipse(100,0,90,20);
  fillcolor('white');
  fill();
  ellipse(-100,0,90,20);
  fillcolor('white');
  fill();

  color('white');
  save();
  rotate(45);
  ellipse(0,-100,20,90);
  restore();
  fillcolor('white');
  fill();

  save();
  rotate(135);
  ellipse(0,-100,20,90);
  restore();
  fillcolor('white');
  fill();

  save();
  rotate(225);
  ellipse(0,-100,20,90);
  restore();
  fillcolor('white');
  fill();

  save();
  rotate(315);
  ellipse(0,-100,20,90);
  restore();

  fillcolor('white');
  fill();

  color('yellow');
  circle(0,0,25);
  fillcolor('yellow');
  fill();
}
