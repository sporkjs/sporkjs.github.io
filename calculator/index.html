Calculator Help
  • Input Boxes
    • There are a variety of input boxes in different dialogs of the calculator. The active entry box is the one outlined in red. When a key is pressed, the result is entered in the active entry box. To activate a box, click on it or touch it.
  • Arithmetic
    • Arithemtic Operations: +, *, -, /
    • Exponents: x^y use a carrot for simple exponents like 3^2 or 2^0.5 for more complicated exponents, use pow(x,y)
    • Other Functions: abs(x), acos(x), asin(x), atan(x), ceil(x), cos(x), exp(x), floor(x), log(x), max(x,y), min(x,y), random(), sin(x), sqrt(x), tan(x)
    • Constants: E, PI
  • Math Button
    • This gives a list of standard mathematical functions such as sqare root, factorial, log, exponential, sin, cos, tan, pow (exponents), permutations, combinations.
    • This also gives the constants E and PI.
  • Stat Button
    • sumstat(i) will give the summary statistics for data listed in List #i.
    • varstat(i,j) will give the parameters for a random variable whose values are in list #i and whose probabilities are in list #j.
    • binompdf(n,p,b) will calculate P(x = b) where x is a random variable with a binomial distribution with n trials and probability of success p.
    • binomcdf(n,p,b) will calculate P(x ≤ b) where x is a random variable with a binomial distribution with n trials and probability of success p.
    • poissonpdf(m,b) will calculate P(x = b) where x is a random variable with a Poisson distribution with mean m.
    • poissoncdf(m,b) will calculate P(x ≤ b) where x is a random variable with a Poisson distribution with mean m.
    • normalcdf(a,b) will calculate P(a < z < b) where z is a random variable with a standard normal distribution.
    • normalcdf(a,b,m,s) will calculate P(a < x < b) where x is a random variable with a normal distribution with mean m and standard deviation s.
    • normalcdf(a) will calculate P(z < a) where z is a random variable with a standard normal distribution.
    • invnorm(p) will calculate a number a so that P(z < a)=p where z is a random variable with a standard normal distribution.
    • tcdf(a,b,df) will calculate P(a < t < b) where t is a random variable with a Student-t distribution with df degrees of freedom.
    • tcdf(a,df) will calculate P(t < a) where t is a random variable with a Student-t distribution with df degrees of freedom.
    • invt(p,df) will calculate a number a so that P(t < a)=p where t is a random variable with a Student-t distribution with df degrees of freedom.
    • Dialogs The normal, binomial, and poisson dialogs will open dialogs to make it easier to enter commands related to calculating probabilities for these distributions.
  • StatPlot Button Open dialogues to draw rough statistical plots.
  • Test Button
    • Note: The buttons activate dialogues. They do not insert editable commands.
    • cimeant will open a dialogue to calculate a confidence interval for a mean using a Student-t distribution.
    • ciprop will open a dialogue to calculate a confidence interval for a proportion.
    • ztest -- tests about one mean knowing the population standard deviation
    • ttest -- tests about one mean not knowing the population standard deviation
    • onepropz -- tests about one proportion using a normal approximation to the binomial distribution
    • twopropz -- tests about two proportions using a normal approximation to the binomial distribution
    • matchedpairs -- tests for pairs of dependent data entered in two lists
    • twomeant -- tests about two independent means
    • chi2GOF -- goodness of fit test. Expected and observed frequencies must be entered in lists.
    • contingency -- contingency table. The table must be entered in adjacent lists beginning with "First List" through "Last List."
    • anova -- ANOVA test. The samples must be listed in adjacent lists beginning with "First List" through "Last List."
    • correlation -- test for linear correlation in pairs of data entered in lists.
    • binomialtest -- tests about one proportion using a binomial distribution rather than a normal approximation to the binomial.
  • List Button
    • This button opens the lists for editing. Most applications of the lists need one number (or arithmetic) expression on each line. Blank lines and extraneous text may wreak havoc.
